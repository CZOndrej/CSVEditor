Překlad:
    - programl lze přeložit spuštěním souboru Makefile ve složce se zdrojovými soubory, poté se vytvoří spustitelný soubor csve.

Spuštění programu:
    - je nutné zadat vstupní a výstupní soubor jako parametr. -i *.csv -o *.csv

Příkazy
    - addcol <sloupec> <radek0> <radek1> <radek2>, ..., <radekN> - Přidá sloupec (je nutné dodržet počet řádků)
    - addrow <sloupec0> <sloupec1> <sloupec2>, ..., <sloupecN> - Přidá řádek (je nutné dodržet počet sloupců)
    - average <nazev-sloupce> - Vypíše průměrnou hodnotu zadaného sloupce.
    - max <nazev-sloupce> - Vypíše největší hodnotu zadaného sloupce.
    - min <nazev-sloupce> - Vypíše nejmenší hodnotu zadaného sloupce.
    - sum <nazev-sloupce> - Vypíše součet hodnot zadaného sloupce
    - exit - Zapíše (upravenou) databázi na zadanou výstupní cestu a ukončí program.
